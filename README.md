# Pehnava – Fashion E-Commerce Website

A simple responsive fashion e-commerce landing page built using HTML, CSS and JavaScript.

## Features
- Product listing
- Search products
- Category filter (Men / Women / Kids)
- Dark mode toggle
- Responsive design
- External purchase links

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Remix Icons
- Google Fonts
